# RetailFlow Dashboard — Project Overview

## Purpose

RetailFlow Dashboard was created to demonstrate practical front-end development skills using React, modern dashboard interfaces and product-oriented UI/UX.

---

## Main Technologies

- React
- Vite
- Tailwind CSS
- Framer Motion
- Recharts

---

## Current Features

- Responsive dashboard
- Client management
- Orders management
- Payment management
- Financial metrics
- Charts and reports
- localStorage persistence
