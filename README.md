# RetailFlow Dashboard

Dashboard SaaS para pequenos negócios controlarem clientes, pedidos, pagamentos e cobranças.

## Stack

- React
- Vite
- Tailwind CSS
- React Router DOM
- Framer Motion
- Lucide React
- Recharts

## Funcionalidades

- Dashboard com métricas e gráficos
- CRUD local de clientes
- CRUD local de pedidos
- CRUD local de pagamentos
- Relatórios simples
- Layout responsivo
- UI dark mode premium
- Dados mockados na V1

## Rodar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
